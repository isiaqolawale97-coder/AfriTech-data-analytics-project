# AfriTech Data Analytics Project

This project analyzes Afritech's business data to generate insights across customer behavior, transaction performance, and social media engagement.

## Tools Used
- Power BI
- SQL
- Excel

## Project Overview

This project includes three business intelligence dashboards built to help Afritech monitor performance and make data-driven decisions.

### 1. Social Media Crisis Analysis
This dashboard monitors engagement across platforms and identifies sentiment trends to detect potential brand crises.

### 2. Customer Data Analysis
This dashboard analyzes customer demographics, regions, and customer types to understand customer distribution.

### 3. Transaction Data Analysis
This dashboard explores revenue trends, product performance, and transaction patterns.
